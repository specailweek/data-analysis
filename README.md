# 商品销售数据分析

## 1. 项目介绍

本项目使用 Python 对商品销售数据进行分析。通过读取 Excel 表格，将商品信息和销售数据合并，计算销售金额，并从月份、商品大类和商品小类等角度进行统计和可视化。

数据文件为 `商品销售数据.xlsx`，主要包括：

- `信息表`：商品编号、商品类别、商品价格等信息。
- `销售数据表`：订单日期、订单数量等销售信息。

## 2. 分析方法

本项目主要使用 `pandas` 处理数据，使用 `matplotlib` 绘制图表。分析步骤如下：

1. 读取商品信息表和销售数据表。
2. 根据商品编号合并两张表。
3. 计算销售金额：销售金额 = 订单数量 × 商品销售价。
4. 按月份统计销售金额，分析每月销售变化。
5. 按商品大类统计销售金额，比较不同类别的销售情况。
6. 按商品小类统计销售金额，观察各小类的销售差异。

## 3. 分析结论

通过分析可以看出，商品销售金额在不同月份之间存在一定波动，不同商品类别的销售表现也不相同。部分商品小类的销售金额较高，对整体销售贡献较大。因此，可以根据销售变化情况调整库存和促销策略。
